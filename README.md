# halaman kecil — untuk zahra

sebuah proyek scrapbook digital premium yang dirancang dengan estetika warm-vintage (ala pinterest) untuk menyimpan kenangan, surat, dan foto berharga secara interaktif. proyek ini dilengkapi dengan musik latar otomatis, efek mengetik teks secara dinamis, dan fitur *interactive tactile narrative lightbox* pada galeri fotonya.

## 📁 struktur berkas

pastikan susunan folder dan berkas kamu terlihat seperti ini di dalam direktori proyek:

```text
proyek-scrapbook/
├── index.html
├── style.css
├── script.js
├── lagu.mp3
└── assets/
    └── images/
        ├── photo1.jpg
        ├── photo2.jpg
        ├── photo3.jpg
        ├── photo4.jpg
        └── photo5.jpg
